# Projeto CulturaShow
Este projeto tem como objetivo criar uma plataforma de eventos culturais, onde os usuários podem encontrar e participar de eventos relacionados à música, teatro, dança e outras formas de arte.

## Requisitos Funcionais (RF)
### Requisitos de usuário (RF-U)
1. RF-U1: O usuário deve ser capaz de visualizar uma lista de eventos culturais disponíveis na plataforma.
2. RF-U2: O usuário deve ser capaz de pesquisar eventos por categoria, localização e data.
