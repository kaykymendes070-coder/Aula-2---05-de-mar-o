# Aula 2 - 05 de março

## Requisitos funcionais para diferentes tipos de usuários
Nesta aula foi discutido os requisitos funcionais para diferentes tipos de usuários do sistema. Os principais tipos de usuários identificados foram:
- **Usuário comum**: Deve ser capaz de criar uma conta, fazer login, visualizar e editar seu perfil, e acessar as funcionalidades básicas do sistema.
- **Administrador**: Deve ter acesso a um painel de controle onde pode gerenciar usuários, eventos e parceiros, visualizar estatísticas do sistema, e configurar as opções avançadas.
- **Parceiro**: Deve ser capaz de criar e gerenciar eventos, visualizar estatísticas de seus eventos, e interagir com os usuários que participam de seus eventos.
O diagrama foi desenhado para ilustrar as interações entre os diferentes tipos de usuários e as funcionalidades do sistema e esta disponível no arquivo `.\docs\usuarios.excalidraw`.
